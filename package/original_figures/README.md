# 原始图表说明

本文件夹用于存放原始论文中的图表。

## 注意事项

由于版权原因，原始论文的图表可能无法直接包含在此项目中。

## 如何获取原图

1. 访问发表期刊网站（Global Change Biology）
2. 下载原始论文：Gutiérrez-Cánovas, C., et al. (2024)
3. 从论文中提取相关图表
